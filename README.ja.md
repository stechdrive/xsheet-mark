# xsheet-mark

動画を再生したまま、上に重ねてアニメーションのタイムシートにメモを書き込める Windows 用オーバーレイ描画ツール。

Premiere Pro、DaVinci Resolve、VLC など任意の動画再生アプリの上に、**ピクセル単位で透過する常に最前面のウィンドウ**として被せて使います。オーバーレイをクリック／描画しても動画側のフォーカスを一切奪わないので、そのままスペースキーで再生／一時停止しながらタイミングをマークできます。

*English: [README.md](README.md)*

## 特徴

- **フォーカス非奪取のオーバーレイ** — ペンでもマウスでもオーバーレイに描画でき、動画側のキーボード制御が効き続ける
- **ペンタブ対応** — Windows Ink 経由で筆圧取得（HUION で動作確認済み、Wacom 系も動作見込み）
- **無限キャンバス** — 任意枚数のタイムシートを並べて読み込み、自由にパン／ズーム
- **ツール別線幅記憶** — ペンと消しゴムそれぞれに独立した線幅設定
- **5 色、3 段階の線幅、ポイント消しゴム**をツールバーから選択
- **Undo / Redo** — 描画・消しゴム・画像ドロップ・画像移動・キャンバスクリア全てを取り消し可能
- **ウィンドウと画像の不透明度を独立調整** — オーバーレイ全体を透かしたり、画像だけ薄くしてメモと比較したり
- **対応画像形式** — PSD、PSB、JPG、PNG、BMP、GIF、TIFF、TGA
- **PSD 書き出し（画像ごと）** — キャンバス上の画像ごとに 1 ファイルずつ PSD を出力。PSD を読み込んだものは元のレイヤー構造を保持したまま `xsheet-mark` レイヤーが最上位に追加される。その他の形式は新規 PSD（画像レイヤー + ink レイヤー）を生成
- **ビューポートキャプチャ** — 別ボタンで、今のビューポートに見えているもの（画像 + ink、現在のズーム／パン）を**背景透明**の 1 枚 PSD として出力。注釈を後で他の素材に合成したい用途向け
- **画像を動かすとメモも追従** — 画像矩形内に描いたストロークが画像と一緒に移動
- **キャンバスをリセット** — 1 クリックで画像とストロークを全クリア（取り消し可）
- **日英 UI** — Windows の表示言語から自動判定

## 動作環境

- Windows 10 または 11（64 ビット）
- 筆圧を使う場合、ペンタブドライバの設定で **「Windows Ink」を有効** にする必要あり（マウスだけなら不要）

## 使い始め方

1. [Releases](https://github.com/stechdrive/xsheet-mark/releases) から `xsheet-mark.zip` をダウンロード
2. 任意の場所に展開し `XsheetMark.exe` を実行
3. 動画再生アプリで再生を開始
4. タイムシート画像を xsheet-mark のウィンドウにドラッグ＆ドロップ（または画像なしで最初から描き始めてもOK）
5. ペンで書き込み — 動画は再生したまま、キーボードショートカットも効き続けます

## 使い方

### ウィンドウ周り

- **上部の黒いバー** — ドラッグでウィンドウを移動
- **—（最小化）** — タスクバーに収納。タスクバーアイコンをクリックで復帰
- **✕（閉じる）** — アプリ終了
- **ウィンドウの縁** — カーソルを近づけるとリサイズカーソルに変化、ドラッグでリサイズ
- **不透明度パネル（右上）** — 2 本のスライダー：
  - *ウィンドウ*（0.2〜1.0）：オーバーレイ全体を透かして下のアプリを見る
  - *画像*（0.0〜1.0）：読み込んだ画像を一括で薄くする（ink はそのまま見える）

### ツールバー（左側）

| アイコン | 操作 |
|---|---|
| ↶ | 直前の描画／消し／画像ドロップ／画像移動／リセットを取り消し |
| ↷ | やり直す |
| ✎ | ペン |
| ⌫ | 消しゴム（ストローク全体ではなく、触れた場所だけを削る。サイズは線幅選択と連動） |
| ✥ | 画像移動 — 画像をドラッグすると、その画像矩形内のストロークも一緒に移動 |
| ● | インクの色（黒／赤／青／緑／黄） |
| ▬ | 線幅（ペン・消しゴムそれぞれが独立に記憶） |
| ⛶ | 全画像がウィンドウに収まるようフィット |
| 💾 | 画像ごとに PSD で書き出し |
| 📷 | 現在のビューポートを 1 枚の PSD としてキャプチャ（背景透明） |
| 🗑 | キャンバスをクリア（取り消し可） |

### キャンバス操作

- **ホイール** — カーソル位置を中心にズーム
- **中クリックドラッグ** または **右クリックドラッグ** — キャンバスをパン
- 画像ファイルをウィンドウにドラッグ＆ドロップで読み込み
- **画像を読み込まなくても**キャンバスに直接描画可能

### 書き出し

ボタン 2 つで使い分け：

- 💾 **Export** — キャンバス上の画像ごとに `<元ファイル名>-marked.psd` を 1 つずつ出力。PSD 入力は元のレイヤー構造を保ったまま、サムネイルと composite preview（Maximize Compatibility の領域）も書き直すのでファイラーのプレビューも注釈入りになる。キャンバスに画像が無い場合はビューポートサイズの 1 枚 PSD にフォールバック
- 📷 **Capture** — 画像の有無に関わらず、現在のビューポートを **背景透明**の 1 枚 PSD に。「透過注釈を撮って後で素材に合成する」用途向け

どちらもフォルダ選択で出力先を指定、ファイル名が衝突した場合は `-2, -3...` の連番で回避。

### ペン筆圧

ペンタブドライバ側の設定で「Windows Ink」を有効にしてください。WPF は Windows Ink／Real-Time Stylus 経由で筆圧を読み取ります。WinTab のみの構成では筆圧が渡りません。
